# 差し替え用ファイル群

このフォルダの中身を、既存の GitHub Pages リポジトリ直下に上書き配置してください。

## 主な差し替え対象
- index.html
- favicon.ico
- favicon-16x16.png
- favicon-32x32.png
- apple-touch-icon.png
- icon-48x48.png
- icon-192x192.png
- icon-512x512.png
- soup-title-icon.png
- soup-icon-1024.png
- site.webmanifest

## 補足
- タイトル先頭アイコンは `soup-title-icon.png` を使用しています。
- ファビコンは、選択いただいた「シンプルなスープアイコン」版に差し替え済みです。
- `original-selected-image.png` は採用元画像です。
